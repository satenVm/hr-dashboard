# HR Dashboard

A responsive HR management dashboard built with Next.js, Tailwind CSS, and Recharts.

## Features
- Employee list with department filtering
- Headcount bar chart by department
- KPI metric cards (headcount, salary, turnover)
- Clean, responsive UI

## Tech Stack
- Next.js 16 (App Router)
- Tailwind CSS
- Recharts
- TypeScript

## Getting Started
npm install
npm run dev
